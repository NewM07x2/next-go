# 概要
このアプリケーションは、タスクを管理するためのシステム(next-go-task)です。  
ユーザーは新規登録、ログイン、タスクの登録・閲覧・編集・削除が可能です。

---

## 開発環境
バックエンド: Go  
フロントエンド: Next.js  
データベース: PostgreSQL  
コンテナ管理: Docker  

---

## 環境構築

以下の手順で環境を構築してください。

### 1. リポジトリのクローン

```bash
git clone <リポジトリのURL>
cd next-go-task
```

### 2. Dockerコンテナの起動

